Changelog

[0.1.0] - Initial Release

Added

· Basic background MenuBar app without Dock icon
· Launch at startup with last saved configuration
· Audio device auto-detection for inputs and outputs
· Multi-output routing system (split from single input to multiple outputs)
· Dynamic Route Mapping system (#A, #B, #C, etc.)
· Mute, volume control (0-100) per route
· 16-band parametric EQ per route (Gain -24 to +24, Freq 0Hz-20kHz, Q 0.4-40)
· EQ preset save/load functionality
· Support for system-wide audio capture via Blackhole 2ch (if installed)
· Fallback to connected audio devices only
· macOS 11+ compatibility for Intel and ARM64
· GitHub Actions build automation
